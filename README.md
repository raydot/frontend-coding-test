## 25. Staircase Problem

The staircase problem is just the fibonacci problem offset by one. Thus:

```javascript
staircase(1) = fibonacci(2) = 1
staircase(2) = fibonacci(3) = 2
staircase(3) = fibonacci(4) = 3
```
